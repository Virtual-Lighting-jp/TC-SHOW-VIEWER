# TC Show Viewer

LTC / MTC / Manual タイムコード同期のキューリスト表示アプリ。

ブラウザだけで動作し、インストール不要。舞台・イベント・コンサートなどのタイムコード運用に。

## 主な機能

- **タイムコード受信** — LTC（オーディオ入力）/ MTC（MIDI）/ 手動入力
- **フレームレート** — 24 / 25 / 29.97 / 30fps 対応。LTC受信時の自動検出あり
- **キューリスト** — 全キューを一覧表示。現在のTCに該当するキューをハイライト
- **M.No（楽曲番号）** — 楽曲ごとのグルーピングと残り時間表示
- **CSV読み書き** — CSVファイルでキューを管理。画面内エディタで直接編集も可能
- **現在時刻表示** — 右パネルに常時表示

## CSV フォーマット

```
M.No, TC, 楽曲尺, テキスト
M1,01:00:00:00,04:30,オープニング
M1,01:00:30:00,,サビ前テロップ
M1,01:02:00:00,,ラストコール
M2,01:05:00:00,03:00,次の曲スタート
```

- **M.No** — 楽曲番号（グルーピング用）
- **TC** — LTCタイムコード（HH:MM:SS:FF）
- **楽曲尺** — 各M.Noの最初の行にだけ記載（MM:SS）
- **テキスト** — 表示する内容

## 使い方

### ブラウザで直接開く（一番簡単）

`TC_Show_Viewer.html` をChromeで開くだけ。

### Windows アプリとして使う

`TC_Show_Viewer.bat` をダブルクリックすると、Chromeがアプリモード（アドレスバー無し）で起動します。

### Windows の .exe を作る

`exe作成.bat` をダブルクリックすると `TC_Show_Viewer.exe` が生成されます。Node.js 不要、Windows標準の .NET コンパイラで作成します。配布は `TC_Show_Viewer.exe` と `TC_Show_Viewer.html` の2ファイルセットで。

## キーボードショートカット

| キー | 動作 |
|------|------|
| `E` | EDIT画面 開閉 |
| `Tab` | サイドバー 開閉 |

## 動作環境

- **Chrome 推奨**（LTC・MIDIはマイク/MIDI権限が必要）
- MTC使用時は Chrome拡張「WebMIDI Browser」推奨
